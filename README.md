# Addiction Game

Game representing Drugs Addiction. This is a group project from Minor Embrace TEC at Fontys Univeristy.

# Tech stack

- Vite
- React.js + TS
- Tailwind CSS
- Eslint, Prettier

# Development

## Install packages

> npm install

## Run development server

> npm run dev

## Build app

> npm run build

## Run Linter

> npm run lint

## Check Formatting

> npm run format

### Auto-fix formatting

> npm run format:fix
